Arcane Innovation Tech (AIX)

The AIX Enterprise Ledger is a fully normalized, transactional management portal engineered to unify internal account security, dynamic inventory workflows, and multi-table financial reconciliations.

Designed with a strict focus on data integrity, the system ensures atomic execution across core business processes while presenting administrators with a clean, highly accessible, and visually optimized control dashboard.

Engineered Stack

This application operates on a decoupled full-stack routing architecture, linking robust client-side validation logic directly to an enterprise-grade backend infrastructure.

  • 🖥️ HTML5 / CSS3 Grid
  • ⚡ Vanilla JS Orchestration
  • 📊 Chart.js Analytics Engine
  • ⚙️ ASP.NET Core Web API
  • 🗄️ MySQL Relational Schema
  • 🔒 SHA-256 Auth Pipeline

Operational Scope

  • User Account Control: Secure cryptographic gateway handling access validation, state tracking, and local recovery token routing.
  • Live Inventory Nodes: Atomic dynamic scalar mapping enabling granular stock surveillance alongside real-time threshold warnings.
  • Relational Processing: Complex multi-table inner joins automating concurrent entries across orders, details, and settlements.
